# KIO Multi-Company Stock Transfer

Manage stock transfers between two different companies inside the same Odoo database.

## Overview

This module provides functionality for managing inter-company stock transfers within a multi-company Odoo environment.

## Features

- Inter-company stock transfer management
- Multi-company inventory synchronization

## Installation

1. Place this module in your Odoo addons directory
2. Update the module list
3. Install "KIO Multi-Company Stock Transfer"

## Requirements

- Odoo 17.0
- Stock module
- Mail module
